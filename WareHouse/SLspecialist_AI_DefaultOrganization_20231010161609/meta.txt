Task:
We are in need of software that can manage income and expenses, supporting both LKR and USD currencies formats. It should include the following features: 1. Login Feature: Users should be able to log in securely. 2. Dashboard: After logging in, users should access a dashboard that provides information about their accounts, contacts, transactions, and analytics. 3. Import and Export Functions: These functions should be easily accessible from the dashboard. 4. Dashboard Components: The dashboard should display cash flow, balance trends, operating profit, and other relevant features with graphical representations. 5. Custom Date Range: Users should have the option to select a custom date range (month, week, year, day) and view all details in the dashboard according to that time frame. 6. Account Management: Users should be able to manually set up accounts with names. 7. Transaction History: The software should allow users to upload transaction history by importing CSV or Excel files. 8. Manual Data Update: Users should be able to manually update data as needed. 9. Machine Learning Feature: The software should include a machine learning feature to predict future financial situations based on previous income and expense data. 10. Dashboard Integration: The machine learning predictions should be visible in the dashboard area.
